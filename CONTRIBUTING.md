# Contributing

Thanks for your interest in improving Screenshot Metadata Mod. Contributions
are welcome and appreciated.

## Quick Start

1. Fork and clone the repository.
2. Create a feature branch.
3. Make small, focused changes.
4. Run relevant checks (see below).
5. Open a pull request with a clear description.

## Development Setup

- Requirements:
  - Java 21
  - Gradle (wrapper included)

Common commands:

```sh
cd screenshotmetadata
./gradlew build
./gradlew runClient
```

## Testing

If you run tests or manual checks, note them in your PR. If you did not run
tests, please say so explicitly.

## Style and Structure

- Follow existing code style and formatting.
- Keep changes minimal and focused.
- Add or update documentation when behavior changes.
- Update `screenshotmetadata/CHANGELOG.md` for user-facing changes.

## Submitting a Pull Request

Please include:

- A clear summary of the change.
- Relevant screenshots or logs if applicable.
- Any compatibility notes (Minecraft/Fabric versions).

## Reporting Issues

Please use the GitHub issue templates and include:

- Minecraft version
- Fabric Loader version
- Mod version
- Logs or error output when relevant
